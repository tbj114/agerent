# AxaltyX

专业级桌面统计软件，功能对标SPSS标准版。

## 功能特点

- 数据管理：导入、导出、转换、清洗数据
- 描述性统计：均值、方差、频率分布等
- 假设检验：t检验、方差分析、卡方检验等
- 回归分析：线性回归、逻辑回归等
- 数据可视化：图表、图形等
- 机器学习：分类、聚类等

## 安装

```bash
pip install -e .
```

## 运行

```bash
axaltyx
```
