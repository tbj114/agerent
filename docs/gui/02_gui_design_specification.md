## 1. 设计语言定义

### 1.1 色彩体系

基于对字节跳动旗下 Arco Design (#165DFF)、Lark (#3370FF)、火山引擎 (#0066FF) 三个产品的实际调研，AxaltyX 采用 Arco Design 色彩体系作为基础，融合 Lark 的柔和3D风格与火山引擎的渐变语言。

#### 亮色模式 (Light Theme)

| Token | 色值 | 用途 |
|-------|------|------|
| `--color-primary` | `#165DFF` | 主操作按钮、选中态、链接 |
| `--color-primary-light` | `#E8F3FF` | 主色浅底、hover 背景 |
| `--color-primary-hover` | `#4080FF` | 主按钮 hover |
| `--color-primary-active` | `#0E42D2` | 主按钮 active |
| `--color-success` | `#00B42A` | 成功状态、正向指标 |
| `--color-success-light` | `#E8FFEA` | 成功浅底 |
| `--color-warning` | `#FF7D00` | 警告状态 |
| `--color-warning-light` | `#FFF7E8` | 警告浅底 |
| `--color-danger` | `#F53F3F` | 错误状态、危险操作 |
| `--color-danger-light` | `#FFECE8` | 错误浅底 |
| `--color-info` | `#168CFF` | 信息提示 |
| `--color-info-light` | `#E8F7FF` | 信息浅底 |

#### 中性色板

| Token | 色值 | 用途 |
|-------|------|------|
| `--color-text-primary` | `#1D2129` | 主文本 |

### 1.2 字体规范

| 层级 | 字号 | 字重 | 行高 | 字间距 | 用途 |
|------|------|------|------|--------|------|
| Display | 36px | Bold (700) | 44px | -0.02em | 启动页标题 |
| H1 | 28px | Bold (700) | 36px | -0.01em | 页面标题 |
| H2 | 24px | Semibold (600) | 32px | 0 | 模块标题 |

**字体族**: `"Microsoft YaHei", "PingFang SC", "Noto Sans SC", sans-serif`

### 1.3 间距规范（8px 网格）

|-------|-----|------|

### 1.4 圆角规范

| Token | 值 | 用途 |
|-------|-----|------|
| `--radius-none` | 0px | 表格单元格 |
| `--radius-small` | 4px | 小型元素、Tag |
| `--radius-medium` | 8px | 按钮、输入框、下拉菜单 |
| `--radius-large` | 12px | 卡片、面板、对话框 |
| `--radius-xlarge` | 16px | 大型面板、模态框 |
| `--radius-full` | 999px | 药丸形按钮（Lark风格） |

### 1.5 阴影规范

| Token | 值 | 用途 |
|-------|-----|------|
| `--shadow-none` | none | 无阴影 |
| `--shadow-sm` | `0 2px 4px rgba(0,0,0,0.04)` | 轻微浮起 |
| `--shadow-md` | `0 4px 12px rgba(0,0,0,0.08)` | 卡片、下拉菜单 |
| `--shadow-lg` | `0 8px 24px rgba(0,0,0,0.12)` | 对话框、浮层 |
| `--shadow-xl` | `0 12px 36px rgba(0,0,0,0.16)` | 模态框 |

### 1.6 图标规范

- 尺寸: 16px / 20px / 24px / 32px
- 风格: 线性图标（2px描边，圆角端点）
- 颜色: 继承当前文本色，或使用语义色
- 来源: 自定义SVG图标集，存放于 `resources/icons/`

### 2.3 自定义标题栏

```
+==================================================================+
| [Logo 24x24] AxaltyX          [最小化] [最大化] [关闭]           |
+==================================================================+
```

- 高度: 32px
- 背景: `--color-bg-1`
- Logo: 24x24px，左侧 12px 间距
- 标题: H4 字号，`--color-text-primary`
- 窗口控制按钮: 右侧排列

- 双击标题栏切换最大化/还原
- 关闭按钮与其它按钮间距 8px

```

**表格规格**:
- 默认: 100 行 x 100 列
- 行号列: 宽度 48px，灰色背景，右对齐
- 列头: 可双击编辑，默认 `VAR00001` ~ `VAR00100`
- 单元格: 14px 字号，等宽字体，居中对齐
- 网格线: `--color-border-light`，1px
- 选中单元格: 蓝色边框 `--color-primary`
- 选中行/列: 浅蓝背景 `--color-primary-light`
- 列宽: 默认 100px，可拖拽调整
- 行高: 默认 28px
- 大文件支持: 虚拟滚动，仅渲染可见区域
- 支持横向/纵向滚动

## 3. 组件库设计

### 3.1 按钮

| 类型 | 样式 | 圆角 | 高度 | 用途 |
|------|------|------|------|------|
| Primary | 实心蓝底白字 | 8px | 32px | 主要操作 |
| Secondary | 白底蓝字蓝边框 | 8px | 32px | 次要操作 |
| Danger | 实心红底白字 | 8px | 32px | 危险操作 |
| Text | 无边框蓝字 | 0px | 32px | 辅助操作 |
| Icon | 仅图标 | 8px | 32px | 工具栏按钮 |
| Pill | 药丸形（Lark风格） | 999px | 32px | 标签/筛选 |

**按钮状态**:
- Normal -> Hover (亮度+10%) -> Active (亮度-10%) -> Disabled (透明度40%)

### 3.2 输入框

```
+----------------------------------+
| Placeholder text                 |  <- 高度 32px, 圆角 8px
+----------------------------------+

+----------------------------------+
| [x] Selected value          [v]  |  <- 下拉框, 右侧清除+箭头
+----------------------------------+

+----------------------------------+
| Line 1                           |
| Line 2                           |  <- 多行输入, 最小高度 80px
| Line 3                           |
+----------------------------------+

+----------------------------------+
| 0.00              [+] [-]       |  <- 数字输入, 右侧步进按钮
+----------------------------------+
```

- 边框: 1px `--color-border`
- Focus: 边框 `--color-primary`，外发光 `0 0 0 2px rgba(22,93,255,0.2)`
- Error: 边框 `--color-danger`
- 内边距: 8px 12px

### 3.3 表格组件

- 行号列: 固定左侧，灰色背景
- 列头: 可编辑，支持排序
- 单元格编辑: 双击进入编辑模式
- 选中态: 蓝色边框
- 多选: Shift+Click 范围选，Ctrl+Click 多选
- 右键菜单: 复制/粘贴/插入/删除
- 虚拟滚动: 大数据量时仅渲染可见行

### 3.4 树形菜单

```
[>] 分析

- 展开图标: 16px，旋转动画
- 缩进: 每级 20px
- 选中态: 蓝色背景 + 左侧指示条
- 叶子节点: 点击触发对应分析对话框

## 5. 响应式策略

### 5.1 最小分辨率

- 最低支持: 1280 x 720
- 推荐分辨率: 1920 x 1080
- 最佳体验: 2560 x 1440

### 5.2 自适应规则

| 分辨率 | 左侧面板 | 右侧面板 | 表格行为 |
|--------|----------|----------|----------|
| >= 1920 | 200px 展开 | 240px 展开 | 正常显示 |
| 1600-1919 | 200px 展开 | 240px 展开 | 正常显示 |
| 1280-1599 | 48px 折叠 | 自动隐藏 | 正常显示 |
| < 1280 | 提示调整分辨率 | - | - |

### 5.3 窗口状态

- 最小窗口: 1024 x 600
- 默认窗口: 1280 x 800
- 记忆上次窗口大小和位置
- 支持多显示器
- 支持最大化、最小化、还原

## 7. 主题切换

### 7.1 亮色主题（默认）
