# AxaltyX Qt6 GUI 模块 API 文档

> 版本: 1.0.0 | 版权: TBJ114 | 模块: axaltyx_gui

---

## 设计约定

- 基于 PyQt6 构建
- 所有 UI 文字通过 axaltyx_i18n 加载，禁止 Python 内硬编码文字
- 信号-槽机制实现事件通信
- 所有 Widget 类继承自对应 PyQt6 基类

---
